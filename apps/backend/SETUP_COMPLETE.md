## рҹ—„пёҸ Database Schema

### Tables Implemented (13 entities)

1. **users** - User accounts (customers, admins, cashiers)
2. **categories** - Product categories (Pret, Octa West 2026, Desire)
3. **products** - Product catalog with sizes, guides, analytics
4. **product_size_inventory** - Size-based stock management
5. **promotions** - Category/product level discounts (registered users only)
6. **promo_codes** - User-redeemable codes (registered users only)
7. **promo_code_usage** - Track code usage per user
8. **orders** - Customer orders (online & POS)
9. **order_items** - Order line items with sizes
10. **payment_transactions** - Payment gateway tracking
11. **customer_addresses** - Saved customer addresses
12. **wishlists** - Product wishlists (registered users only)
13. **product_analytics** - View/click tracking for analytics

### Key Database Features

вң… Size Guide as HTML table (not image)  
вң… Barcode auto-generated by system (not admin/user)  
вң… Promotions require login (registered users only)  
вң… Promo codes require login (registered users only)  
вң… Guest tracking via session_id  
вң… Full indexing for performance  

## рҹ”җ Authentication

- **JWT Strategy** - Bearer token authentication
- **Access Token** - 15 minutes expiration
- **Refresh Token** - 7 days expiration
- **Password Hashing** - bcrypt (10 rounds)
- **Protected Routes** - All admin/user routes require JWT

## рҹ“Ұ Core Services

### AuthService
```typescript
- hashPassword()          // Hash password with bcrypt
- comparePasswords()      // Verify password
- generateAccessToken()   // Create JWT access token
- generateRefreshToken()  // Create refresh token
- verifyRefreshToken()    // Validate refresh token
```

### ProductsService
```typescript
- findAll()              // List products with filtering
- findById()             // Get product details
- findBySku()            // Search by SKU
- trackView()            // Track product analytics
```

### OrdersService
```typescript
- generateOrderNumber()   // Auto-generate ORD-YYYYMMDD-XXXX
- findAll()              // List orders
- findById()             // Get order details
- create()               // Create new order
- addOrderItems()        // Add items to order
- createPaymentTransaction()  // Track payment
```

### InventoryService
```typescript
- getInventoryByProduct()    // Get all sizes for product
- getInventoryBySize()       // Get specific size stock
- updateInventory()          // Update quantity
- reserveInventory()         // Reserve for pending orders
- releaseReservedInventory() // Release reserved stock
```

### PromotionsService
```typescript
- getActivePromotions()           // Registered users only
- validatePromoCode()             // Validate code for use
- createPromotion()               // Admin create promotion
- trackPromoCodeUsage()           // Track usage per user
- getUserPromoCodeUsageCount()    // Check per-user limit
```

## рҹҡҖ API Architecture

### Base URL
```
http://localhost:3000/api/v1
```

### Module Endpoints (Scaffolded)

#### Authentication
```
POST   /auth/register      // Register user
POST   /auth/login         // Login
POST   /auth/refresh       // Refresh token
POST   /auth/logout        // Logout
```

#### Products
```
GET    /products           // List all products
GET    /products/:id       // Get product details
POST   /products           // Create (admin)
PUT    /products/:id       // Update (admin)
DELETE /products/:id       // Delete (admin)
POST   /products/:id/track // Track view
```

#### Categories
```
GET    /categories         // List all categories
GET    /categories/:slug   // Get by slug
POST   /categories         // Create (admin)
PUT    /categories/:id     // Update (admin)
DELETE /categories/:id     // Delete (admin)
```

#### Orders
```
GET    /orders            // List orders
GET    /orders/:id        // Get order details
POST   /orders            // Create order
PUT    /orders/:id/status // Update status
POST   /orders/:id/cancel // Cancel order
```

#### Inventory
```
GET    /inventory/product/:productId  // Get product stock
PUT    /inventory/:id                 // Update stock
```

#### Promotions
```
GET    /promotions/active              // Active promotions (JWT required)
POST   /promo-codes/:code/validate     // Validate code (JWT required)
POST   /promo-codes                    // Create code (admin)
```
